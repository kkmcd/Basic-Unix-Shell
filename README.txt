To compile
-----------
gcc shell.c

To execute
-----------
./a.out

Salient Features of mySHELL
----------------------------
1. Runs in an infinite loop, prints a command prompt, accepts a command,
   executes the command, and prints the prompt for the next command.
2. Handles foreground as well as background commands
3. Handles internal commands like cd,exit,logout,pwd,whoami,cd..
4. Handles any linux external commands
5. Handles single level of I/O redirection(<,>,|)
6. Keeps track of environmental variables like PATH, PRESENT_WORKING_DIRECTORY,
   USERNAME, SYSTEM.
